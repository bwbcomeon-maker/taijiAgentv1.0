太极 Agent 麒麟终端打包交付

源码包：taiji-agentv1.0-kylin-build-src-0eb1460.tar.gz
源码 commit：0eb1460ba714a68dc61c2c9753524efe5f941393
目标系统：Kylin V10 SP1 / Debian-like / x86_64 amd64 / glibc 2.31
交付形态：目标终端本地构建 amd64 DEB，然后本机安装验证
安装脚本：已支持同版本重装、apt 降级覆盖和 hold 包覆盖，不需要手动卸载旧 taiji-agent。
旧版处理：安装前自动停止并禁用旧服务，清理旧进程，彻底删除旧 hermes-bwb WebUI 版系统安装，不保留两个 Agent 并存运行。
数据边界：旧 `/opt/taiji-agent`、旧配置、旧服务、旧入口会被删除，不再备份旧模型 Key、微信 token 或历史会话；普通用户家目录下的新版用户态目录不删除。
启动修复：桌面启动和 taiji 命令已改为通过 `venv/bin/python -m hermes_cli.main` 启动 Agent，不再依赖 `venv/bin/hermes` 的绝对 shebang；交付目录可放在带空格或中文路径下。
配置修复：安装包内置非密默认产品配置，启动时自动同步菜单显隐、默认模型和首页控件；不会内置或覆盖模型 Key。
窗口修复：Linux 桌面端隐藏应用菜单栏，不再显示“太极 Agent / 视图”，保留麒麟原生窗口按钮。
诊断能力：新增 `taiji-agent-diagnose` 和 `03_目标终端_导出诊断报告.sh`，可导出脱敏诊断报告。

注意：本交付文件夹不包含模型 API Key、微信 token、企业微信 Secret、服务器地址或私钥。
