状态判定规则：

- 存在 P0：状态必须是“未完成”或“阻塞”。
- 存在 P1：状态必须是“未完成”，除非已经修复。
- 存在未验证项目：不得声称该项目通过。
- 没有真实浏览器测试：必须在“未验证项目”中写明。
- 没有截图：必须在“未验证项目”中写明。
- 没有可访问性自动化：必须在“未验证项目”中写明。
- 没有视觉回归：必须在“未验证项目”中写明。
