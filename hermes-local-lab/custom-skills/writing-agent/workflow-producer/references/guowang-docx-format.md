# 国家电网风格 Word 文档格式规范

## 用途
当用户要求以国网模板输出 Word 文档时使用。适用于深度文章、研究报告、行业分析等类型。

## 页面设置
- 上边距: 2.54cm
- 下边距: 2.54cm
- 左边距: 3.17cm
- 右边距: 3.17cm

## 字体规范
| 元素 | 字体 | 字号 | 样式 |
|------|------|------|------|
| 正文 | 仿宋 | 12pt | 首行缩进 0.74cm |
| 一级标题 | 黑体 | 22pt | 加粗, 颜色 #1B5E20 |
| 二级标题 | 黑体 | 16pt | 加粗, 颜色 #2E7D32 |
| 三级标题 | 黑体 | 14pt | 加粗, 颜色 #388E3C |
| 封面标题 | 黑体 | 28pt/22pt | 加粗, 居中, 颜色 #1B5E20 |
| 封面副标题 | 楷体 | 14pt | 居中, 颜色 #666666 |
| 封面日期信息 | 仿宋 | 12pt | 居中, 颜色 #666666 |

## 颜色方案
- 主色: #1B5E20 (深绿, 国网标志性绿色)
- 辅助色: #2E7D32 (中绿)
- 细节色: #388E3C (浅绿)
- 中性色: #666666 (灰色, 用于元信息)

## 文档结构
1. 封面页（标题、副标题、版本号、日期）
2. 备选标题方案（如有）
3. 正文（分部分，每部分可另起一页）
4. 配图建议
5. 发布建议

## 段落格式
- 正文段落: 首行缩进 2 字符（约 0.74cm）
- 标题段落: 段前 18pt，段后 6pt
- 列表项: 不加首行缩进，加粗前缀

## python-docx 实现要点
```python
from docx import Document

from docx.enum.text import WD_ALIGN_PARAGRAPH
from docx.oxml.ns import qn

run.font.name = '仿宋'
run._element.rPr.rFonts.set(qn('w:eastAsia'), '仿宋')
```
