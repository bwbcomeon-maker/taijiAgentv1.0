# 会议纪要模板 JSON 生成要求

你需要把用户已有成果物整理成符合 `schema.json` 的会议纪要 JSON，只输出 JSON，不输出解释文字。

## 字段要求

- `meeting`：会议名称、时间、地点、主持人、记录人。
- `attendees`：参会人员列表，尽量补充角色或部门。
- `topics`：会议讨论议题，每个议题包含标题和摘要。
- `decisions`：会议形成的明确决议，每条包含事项和责任人。
- `actionItems`：会后待办事项，每条包含任务、责任人、截止日期和状态。

## 输出约束

- 只输出一个合法 JSON 对象。
- 不要输出 Markdown 代码块。
- 缺失信息用“待确认”标注，不要编造具体姓名、日期或结论。
